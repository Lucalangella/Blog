# Liminality

> A collective exploring the threshold between what is and what could be.

## About

Liminality is a blog by a collective of developers and designers exploring the intersection of:

- **Artificial Intelligence & Machine Learning**
- **Spatial Computing** (AR/VR/Vision Pro)
- **Design & Innovation**

We exist in the liminal space—the threshold between disciplines, technologies, and ideas. Our articles explore emerging technologies and their potential to transform how we create, interact, and think.

## Getting Started

This blog is built with [Jekyll](https://jekyllrb.com/) using the [fast_template](https://github.com/fastai/fast_template).

### Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`

### GitHub Pages

To deploy this blog:

1. Go to **Settings → Pages**
2. Select branch `master` and root directory
3. Your blog will be live at `https://lucalangella.github.io/Blog`

## Writing Posts

Create new posts in the `_posts` directory following this naming convention:

```
YYYY-MM-DD-title-of-post.md
```

## Contributing

Interested in contributing to Liminality? We welcome thoughtful articles at the intersection of technology, design, and innovation. Open an issue to propose a topic or submit a pull request.
